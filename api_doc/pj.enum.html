PJ.Enum

2.0
pj.enum.js, pj.enum.min.js

Enumeration Class Construct used to create an enum-like structure.

Adapted from Node-Enum by Tolgaek. (https://github.com/tolgaek/node-enum)

Requirements

  • Core Module

Constructor

Enum(String ...)
2.0

Constructs new Enumeration object.

Example usage: var colors = new PJ.Enum("RED", "WHITE", "BLUE");

Any non-string parameter is ignored. Duplicate names are ignored. Names are case-senstive, so "RED", "red", "ReD", etc will all be separate enumeration constants.

string...variable argument list of strings as the names for each Enum Constant.
PJ.IllegalArgumentExceptionthrown if an invalid identifier is given or if an identifier is given twice.
