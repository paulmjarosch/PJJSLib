PJ.TruthTable

4.0
pj.truthtable.js, pj.truthtable.min.js

Evaluates a boolean expression and generates a truth table.

Requirements

  • Core Module

Constructor

TruthTable(Stringexpression, Numbervar_count)
4.0

Constructs a new truth table.

Stringexpressionexpression to evaluate. Must be properly formatted.

All variables must use v as the variable and be in array notation. For example: a AND b would have to be supplied as: v[0] && v[1], where v[0] represents a and v[1] represents b.

Stringvar_countthe number of variables

Methods

ArraygetResults()

Get the raw results array.

4.0
Array

This array is a multi-dimensional array with all the values necessary to populate the truth table. The last element in each row is final evaluated value for that row.

StringgenerateHTML(Objectoptions)

Get the results as an html table.

4.0
ObjectoptionsThe options
  • Stringoptions.idthe table id. Defaults to "truthtable".
  • Booleanoptions.shorthandflag to indicate the use of shorthand boolean values (T/F) instead of (TRUE/FALSE). Defaults to false.
String

HTML code layout with no styling.
