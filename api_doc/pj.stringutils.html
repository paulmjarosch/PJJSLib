Stringpad(Stringstr[, Numberlen[, Stringch[, Numberopt]]])

Pads a string with the specified character until the specified length is reached.

If the length of the string is greater than or equal to len, then the original string is returned.

4.0
Stringstrthe to string to pad. Defaults to empty string.
Numberlenthe total length of the string after padding. Defaults to length of string.
Stringchthe character (or string) to pad with. Defaults to empty space ( ).
Numberoptthe direction to pad.One of the following PJ.StringUtils.PAD_BOTH, PJ.StringUtils.PAD_LEFT, PJ.StringUtils.PAD_RIGHT. Defaults to PJ.StringUtils.PAD_BOTH.
String

The padded string.

ArrayparseQueryString(Stringqa)

Parse Query String into array of key / value pairs.

If there are multiple keys with the same name, the value for that key will be an array of values. This can process multiple keys names with and without brackets..

4.0
Stringqsthe query string to process. This can include the preceeding question mark symbol. This symbol will be ignored.
Number

Array of key/value pairs.
